Detailed Description

Per-sample metadata.

Note that the first three fields are equal to the ones in SampleDefault

Field Documentation

bitrate_t Sample::duration

The duration of each Sample, in increments defined by Manifest::tick. If this field is not present, its implicit value is the value of SampleDefault::duration. Filled from SampleDuration field.

flags_t Sample::settings

The Sample flags. If this field is not present, its implicit value is SampleDefault::settings.

bitrate_t Sample::size

The size of each Sample, in bytes. If this field is zero its implicit value is SampleDefault::size. Filled from SampleSize field.

bitrate_t Sample::timeoffset

The Sample Composition Time offset of each Sample. This field MUST be present if and only if the SampleCompositionTimeOffsetPresent flag is set. Filled from SampleCompositionTimeOffset field.
