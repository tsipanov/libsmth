bool SMTH_finalizelist ( DynList *  list  ) 

Throws away empty slots and sigils the array.

After a call to finalizelist(), DynList::list can be assigned to any pointer or reused. This function is useful to save memory by eliminating unused pointer slots, but it is not necessary to call it before any operation on the list. The list of pointers is closed by a NULL pointer.

Warning:
Finalizing an uninitialised array (i.e. not processed via preparelist() may have unexpected results.
Parameters:
list The list to be finalized.
Returns:
true on success or false.
