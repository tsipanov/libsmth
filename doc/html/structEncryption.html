Detailed Description

Holds the encryption metadata for the samples parsed from SampleEncryptionBox.

Field Documentation

uuid_t Encryption::id

A UUID that identifies the key used to encrypt Samples.

EncryptionType Encryption::type

The algorithm used to encrypt each Sample. Filled from AlgorithmID

count_t Encryption::vectorno

The number of instances of the InitializationVector field in the SampleEncryptionBox field, filled from SampleEncryptionBoxSampleCount field

byte_t* Encryption::vectors

The Initialization Vector for each Sample. MUST be repeated exactly SampleEncryptionBoxSampleCount times. Filled from InitializationVector field. There is no way to tell where one vector ends and another one starts but relying on the vectorsize and vectorno fields.

byte_t Encryption::vectorsize

The size of the InitializationVector field, in bytes. Allowed values are 0x08 and 0x10
