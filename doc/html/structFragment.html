count_t Fragment::index

An ordinal number for the Fragment in the Track timeline.

The value for a Fragment later in the timeline must be greater than for a Fragment earlier in the timeline, but ordinal values for consecutive Fragments are not required to be consecutive.

count_t Fragment::sampleno

The number of Samples in the Fragment, filled from Trun::SampleCount

Sample* Fragment::samples

Per-field settings from TrunBox, repeated exactly SampleCount times.

flags_t Fragment::settings

The value of the SampleFlags field for the first Sample. This field may be 0x00000000: in that case, program should check per-sample settings and default settings. Filled from FirstSampleFlags
