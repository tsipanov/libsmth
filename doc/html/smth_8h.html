enum SMTH_setting

Enumerates the settings that can be retrieved with SMTH_getinfo.

Unless otherwise stated, all values are returned as 32 bit unsigned int Metadata specific to a certain type of stream (e.g. AUDIO), will be zeroed if active stream is not of the appropriate type.

Enumerator:
SMTH_BITRATE 

The current average bitrate, in bps

SMTH_AUDIO_PACKET_SIZE

SMTHh SMTH_open ( const char *  url,
const char *  params 
)

Opens an url for a Smooth Stream and registers a handle, which will be used to fetch data with subsequent calls to SMTH_read.

Parameters:
url The url from which to retrieve the Smooth Stream
params Optional GET params to make the request (e.g. authentication codes, pages, etc...), as an urlencoded string.
Returns:
A pseudofile handle to perform subsequent calls, or NULL
