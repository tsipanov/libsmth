void SMTH_disposemanifest ( Manifest *  m  ) 

Disposes properly of a Manifest struct.

Programmers are advised to use this function to free a Manifest struct, avoiding the direct use of free(). NEVER TRY TO DISPOSE OF A FRAGMENT if parsemanifest() did not return MANIFEST_SUCCESS.

Parameters:
m The manifest to be destroyed.
