Handle *  handle,

Gets the specified setting value for currently parsed stream and stores it into destination. If an invalid setting is requested, dest is nulled.

Warning:
Unless what is SMTH_STREAMS_NO, a track number (as a size_t) must be passed before the pointer to the receiving variable. If not, the call will not work (and may possibly lead to a crash).
Parameters:
what The setting to retrieve
handle The handle to be tested
[stream] The index of the stream to be tested
[dest] Pointer to an appropriate memory area to store the requested setting

int  stream,
Handle *  handle

Reads at most size bytes from Stream stream into buffer using Handle h.

Read past the end of a fragment will return 0, and subsequent reads will return the next chunk. The input stream is empty when two or more subsequent calls return a 0 value.

Returns:
the number of bytes effectively read (0, in case of error or EOS)
