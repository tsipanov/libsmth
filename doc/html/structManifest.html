Detailed Description

Holds the manifest data of the opened stream.

Field Documentation

EmbeddedData* Manifest::armor

Opaque data that can use to enable playback for authorized users

uuid_t Manifest::armorID

A UUID that uniquely identifies the Content Protection System. For instance: {9A04F079-9840-4286-AB92E65BE0885F95}

tick_t Manifest::duration

The duration of the content, measured in ticks, as indicated by the value of the Manifest::tick field.

length_t Manifest::dvrwindow

The length of the DVR window, measured in ticks. If this field is omitted for a live presentation or set to 0, the DVR window is effectively infinite. This field must not appear in on-demand presentations.

bool Manifest::islive

Whether the presentation is a live streaming or an on-demand.

count_t Manifest::lookahead

Size of the server buffer, in number of fragments. This field is set to 0 for on-demand content.

Stream** Manifest::streams

Pointer to the streams array (NULL terminated).

tick_t Manifest::tick

The time scale of the Manifest::duration attribute, specified as the number of increments per second. The default value is 10000000.

chardata** Manifest::vendorattrs

A set of vendor specific attrs, as a sequence of key/name, NULL terminated.
