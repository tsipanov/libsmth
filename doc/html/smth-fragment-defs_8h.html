#define XXX_SKIP_4B_QUIRK
Value:
if (boxsize < 9) \
    {   fseek(root->stream, sizeof(word_t), SEEK_CUR); \
        boxsize -= sizeof(word_t); \
    }

If there are less than 8 bytes remaining in the Box, skips 4B: they are certainly trailing or unknown bytes.

No box is shorter than 9B. This is an orrible quirk, specifically crafted for parsemfhd and parsetfhd, and should been removed as soon as the function of undocumented fields is discovered.
