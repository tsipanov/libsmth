Field Documentation

chardata** Track::attributes

A set of attributes as a NULL terminated array that identify the Track. even slots are keys, odd slots are values.

flags_t Track::audiotag

A numeric code that identifies which media format and variant of the media format is used for each sample in an audio track.

bitrate_t Track::bitrate

The average bandwidth consumed by the track, in bits per second (bps). The value 0 may be used for tracks whose bit rate is negligible relative to other tracks in the presentation.

unit_t Track::bitspersample

The sample Size of an audio track

unit_t Track::channelsno

The Channel Count of an audio track

char Track::fourcc[MANIFEST_TRACK_FOURCC_SIZE+1]

A four-character code that identifies which media format is used for each sample.

hexdata* Track::header

Data that specifies parameters specific to the media format and common to all samples in the track. The meaning of each sequence is correlated to the FourCC type, as follows:

  • "H264": hex-coded string of the following byte sequence: x00 x00 x00 x01 SPSField x00 x00 x00 x01 SPSField SPSField contains the Sequence Parameter Set (SPS). PPSField contains the Slice Parameter Set (PPS).
  • "WVC1": hex-coded string of the VIDEOINFOHEADER structure.
  • "AACL": should be empty.
  • "WMAP": WAVEFORMATEX structure, if the AudioTag field equals "65534", and should be empty otherwise.
  • other: the format of the CodecPrivateData field is vendor-extensible. Registration of the FourCC field value with MPEG4-RA, can be used to avoid collision between extensions.
count_t Track::index

An ordinal that identifies the track and MUST be unique for each track in the stream. The Index should start at 0 and increment by 1 for each subsequent track in the stream.

ScreenMetrics Track::maxsize

The maximum size of a video sample, in pixels.

unit_t Track::nalunitlength

The number of bytes that specify the length of each Network Abstraction Layer (NAL) unit. This field SHOULD be omitted unless the value of the FourCC field is "H264". The default value is 4.

bitrate_t Track::packetsize

The size of each audio Packet, in bytes.

bitrate_t Track::samplerate

The Sampling Rate of an audio track

chardata** Track::vendorattrs

A set of vendor specific attrs, as a sequence of key/name, NULL terminated.
