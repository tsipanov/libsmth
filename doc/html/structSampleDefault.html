Detailed Description

Holds the default sample metadata parsed from the TfhdBox.

Field Documentation

offset_t SampleDefault::dataoffset

The offset, in bytes, from the beginning of the MdatBox field to the Sample field in the MdatBox field. Filled from BaseDataOffset field.

tick_t SampleDefault::duration

The default duration of each Sample, in increments defined by the TimeScale for the Track. Filled from DefaultSampleDuration field

count_t SampleDefault::index

The ordinal of the Sample description for the Track that is applicable to this Fragment. This field should be omitted. Filled from SampleDescriptionIndex field.

flags_t SampleDefault::settings

The default value of the SampleFlags field for each Sample. If it is zero, its implicit value is Fragment::settings. Filled from DefaultSampleFlags field.

bitrate_t SampleDefault::size

The default size of each Sample, in bytes. Filled from DefaultSampleSize field.
